# Smart Medicine Alerter

## 📌 Overview
A smart medicine reminder system built using Arduino that alerts users
to take medicines at scheduled times using buzzer and display.

## 🎯 Objective
To help users remember to take medicines on time.

## ⚙️ Technologies Used
- Arduino
- Buzzer
- LCD Display
- Tinkercad Simulation

## 🧠 Working Principle
The Arduino is programmed with time intervals. At scheduled times,
it activates a buzzer and displays a reminder message.

## 🔌 Components Used
- Arduino
- Buzzer
- LCD Display
- RTC Module (if used)

## 📸 Output / Results
- Alerts user at correct time
- Successful reminder system
